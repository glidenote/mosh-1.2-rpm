# mosh-1.2-rpm

make rpm and install [Mosh](http://mosh.mit.edu/).

## How to use

   git clone git://github.com/glidenote/mosh-1.2-rpm.git
   cd mosh-1.2-rpm
   sudo sh build.sh

## What it does

 * Builds
 * Installs
 * Overwrites/upgrades your currently installed mosh package

## Distro Support

 * CentOS 6.x x86_64
 * Scientific Linux 6.x x84_64
